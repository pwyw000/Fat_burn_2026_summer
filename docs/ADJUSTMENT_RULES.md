# 每周调整规则

每周日做一次。用 Withings **本周有效称重平均值**（至少 3–4 次）。结论写入 `logs/weekly_review.csv`。

## 硬约束（任何周都优先）

1. **体重地板 160 lb**：周均或连续 3 日晨重 ≤ 162 lb 开始预警；**< 160 lb 立即** +150–250 kcal（加米饭/水果），停止扩大缺口。  
2. **2 个月体脂目标 ≈ 12%**：用周均体脂%趋势判断，不看单日噪声。  
3. 一次只改 **一个主旋钮**。

## 1. 先算三个数

相对上周均值（第一周相对开局 178.2 lb / 体脂 20.1%）：

| 符号 | 含义 |
|---|---|
| ΔWeight | 本周均体重 − 上周均体重（lb） |
| ΔFat% | 本周均体脂% − 上周均体脂% |

## 2. 决策表

| 情况 | 体重 | 体脂% | Week+1 |
|---|---|---|---|
| A. 理想 | 仍 ≥ 162 且周降 ≤ 1.5 lb | 下降 | 保持 |
| B. 体脂不动 | ≥ 162，周变 ±1.5 lb | \|ΔFat%\| < 0.2 | 零食再紧或 −150 kcal；或有氧 +1 |
| C. 变胖趋势 | 升 > 1.5 lb | 升或持平 | 减加工零食与主食 20–30 g 熟重；蛋白不变 |
| D. 掉秤太快 / 近地板 | 周降 > 2 lb 或近 160 | 任意 | **+150–250 kcal**；阻力不减；蛋白 160–180 g |
| E. 脂降体重也掉 | 周降 1.5–2.5 lb 但仍 ≥ 162 | 下降 | 可接受；连续两周则略加碳水 |
| F. 恢复差 | 任意 | 任意 | 先减训练量；饮食先不动 |

## 3. 三档节奏（体脂平台再用）

| 档位 | 何时 | 改动 |
|---|---|---|
| 保守 | 忙、恢复差、体重近 160 | 只砍甜饮零食 |
| 适中 | 平台 1–2 周且体重 ≥ 165 | −150 kcal 或 +1 轻松有氧 |
| 激进 | 平台 ≥3 周、恢复好、体重 ≥ 168 | −250 kcal + 有氧；阻力仍 4–5 天 |

激进档连续不超过 2–3 周；体重跌破 165 后退出激进。

## 4. 有效标准（滚动看）

- 体脂% 朝 **12%** 下降  
- 体重始终 **≥ 160 lb**  
- 阻力多数周 ≥4/5 次  
- 执行率 < 70% 时先修执行（**每日上传 Muscle Booster Nutrition 截图**、蛋白是否达标），不上激进缺口
