# Week 1 基线计划

**档案：** 男 · 41 岁 · 6'0" · 178.2 lb · BMI ≈ 24.2 · 体脂 20.1%  
**目标：** 维持 BMI（体重大致持平）· 降低体脂%  
**成功怎么看：** Withings 周均值 → 体重/BMI 稳，体脂% 下降或趋势向下

设备：Withings = 结果；Muscle Booster = 阻力；Garmin = 有氧；WHOOP = 恢复。  
记录方式：用户只上传截图到 `logs/` 分类文件夹；Agent 从截图写入 `daily_log.csv` / `weekly_review.csv`。

---

## 1. 数字目标

| 指标 | Week 1 目标 |
|---|---|
| 体重 / BMI | 周均体重 **176–180 lb**（BMI ≈ 23.9–24.4） |
| 体脂% | 方向向下即可 |
| 每日蛋白 | **约 160 g** |
| 每日热量 | **约 2,200–2,400 kcal**（接近维持） |
| 阻力训练 | **5 天** · 每次 **10–20 分钟** |
| 有氧 | **2–3 次** · 每次 **20–30 分钟** 轻松有氧 |
| 称重 | 每周 **≥4 天** 早晨空腹同条件 |

Week 1 是基线校准：执行 + 干净数据，不猛砍热量。之后按 `ADJUSTMENT_RULES.md` 滚动调整。

---

## 2. 饮食原则

1. 每餐有明确蛋白（鸡、火鸡、鱼、蛋、希腊酸奶、cottage cheese、蛋白粉）。
2. 主食可控：米饭/面/面包比平时少约 1/4；蔬菜和蛋白先吃。
3. 饮料不带糖：水、黑咖啡、无糖汽水。
4. 方便食品可以，优先「加热即食 + 蛋白 ≥ 20 g/份」。

### 一日结构（可打乱）

| 餐次 | 怎么组 | 蛋白大约 |
|---|---|---|
| 早餐 | 蛋 2–3 个或希腊酸奶大杯 | 30–40 g |
| 午餐 | 烤鸡/火鸡/金枪鱼 + 蔬菜 + 少量碳水 | 40–50 g |
| 晚餐 | 家里肉菜正常吃；主食略减 | 40–50 g |
| 加餐 | 蛋白粉 / cottage / 牛肉干 / 酸奶 | 25–35 g |

### 老人做饭

- 照常吃肉和菜；米饭/面比平时少半碗到 1/3。
- 肉少则饭后补蛋白粉或 cottage。
- 外食大餐后第二天回模板，不要惩罚性饿一天。

---

## 3. 训练

- **阻力**：Muscle Booster 10–20 分钟，目标每周 5 次；动作太轻松则下周升难度。
- **有氧**：Garmin 轻松有氧 2–3 次；不追求 HIIT。
- **WHOOP 红/黄且练不动**：优先减训练量（情况 F），饮食先不动。

1. 昨日/今晨数据摘要（体重、体脂、恢复、训练、蛋白）

5. 风险提醒（睡眠、外食、连续平台期等）
